## 如果本项目对您有帮助，欢迎点击Star一下！！！

#### 简介
* 基于Laravel开发，支持markdown语法的博客。

#### 功能介绍
* markdown文章编辑器
* 文章发布
* 时间轴
* 评论功能（3.25添加评论功能，评论回复邮件提醒功能）
* 浏览数统计
* 文章分类
* 文章标签
* 导航栏自定义
* 文章评论
* 关键词
* 搜索功能
* 系统基本设置
* 友情链接
* 文件上传管理
